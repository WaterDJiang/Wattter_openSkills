# Brief Gate · 生成前确认

当用户没有明确给出内容组织、视觉风格、颜色主题、模板预设或动效模式时，生成 deck 前必须先给一张确认卡。不要直接进入制作，除非用户明确说“直接生成 / 你决定 / 不用确认”。

## 何时必须确认

任一项缺失就触发 brief gate：

- 内容组织方式不明确
- 风格未指定：`editorial` / `swiss`
- 颜色主题未指定：预设主题或自定义品牌色
- 模板预设 / 页面组织未指定
- 动效模式未指定
- 分享场景、受众、时长明显缺失且会影响页数或表达节奏

如果用户给了完整大纲和强约束，也不要完全跳过；先用 4-6 行总结你将采用的配置，让用户确认或修改。

## 确认卡格式

用简短选项，不要长篇解释。默认给推荐项，并说明依据。对用户展示时使用中文风格名，内部 style id 只放在括号里，避免 `editorial` / `swiss` 这种工程代号显得随意。

```markdown
生成前我需要你确认 5 项；如果不想细选，直接回复“按推荐生成”。

1. 内容组织：
   A. 观点叙事（推荐：适合分享/演讲）
   B. 方法论拆解
   C. 数据报告
   D. 产品/方案发布
   E. 作品/案例展示

2. 视觉风格：
   A. 杂志叙事型（editorial，推荐：适合观点、故事、个人表达、发布会开场）
   B. 瑞士信息型（swiss，适合数据、产品、方法论、商业汇报）
   C. Linear 科技型（linear，适合 SaaS、AI 工作流、产品路线图、工程团队）

3. 颜色主题：
   A. 按内容自动推荐（我会给出主题名、主色号和选择理由）
   B. 使用预设主题：<列出 3-5 个最相关主题，必须带主色号>
   C. 使用自定义品牌色：请给 #RRGGBB

4. 模板/页面结构：
   A. minimal · 6 页
   B. presentation · 10 页（推荐：通用演讲）
   C. data-report · 8 页
   D. image-showcase · 8 页
   E. 自定义页数/结构

5. 动效：
   A. subtle · 微动（推荐）
   B. static · 静态
   C. cinematic · 沉浸（仅 editorial）

确认后我会用 `YYYYMMDDHHmm_内容主题_模板主题色动效` 命名输出目录，例如：
`202606181530_AI课程合作介绍_瑞士克莱因蓝电影`
```

## 视觉风格展示名

对用户展示时使用下列表达；生成和校验时仍使用内部 id。

| 展示名 | 内部 id | 判断信息 | 不适合 |
|---|---|---|---|
| 杂志叙事型 | `editorial` | 更像电子杂志、演讲开场、观点文章；适合故事、洞察、个人表达、品牌气质 | 大量数据表、严肃经营分析 |
| 瑞士信息型 | `swiss` | 更像国际主义海报和咨询报告；适合数据、方法论、产品结构、商业汇报 | 情绪化故事、需要大量图像氛围的内容 |
| Linear 科技型 | `linear` | 更像深色 SaaS 产品界面；适合 AI 工作流、issue board、roadmap、agent command、工程团队发布 | 人文故事、生活方式、需要温暖纸感的内容 |

不要只给用户看 `editorial` / `swiss` / `linear`，除非用户已经熟悉内部命名。

给颜色选项时必须带主色号和判断信息。不要只列主题名。

## 推荐规则

- 只有主题/想法，没有完整内容：默认先推荐“杂志叙事型 + presentation + subtle”，但必须确认。
- 数据、方法论、产品分析、业务汇报：推荐“瑞士信息型 + 克莱因蓝 `#002FA7` 或 银行红 `#BA0C2F` + subtle”。
- SaaS、AI 工作流、路线图、工程团队发布：推荐“Linear 科技型 + Linear 深紫蓝 `#5E6AD2` 或 极光紫 `#8B5CF6` + subtle”。
- 品牌发布、生活方式、强视觉表达：推荐“杂志叙事型 + 沙丘 `#B8860B` / 牛皮纸 `#8B5E3C`”，或“瑞士信息型 + 爱马仕橙 `#F37021` / 华伦天奴粉 `#E4007F`”，视内容而定。
- 用户提供明确品牌色：不要登记为长期主题；用 `scripts/make-custom-theme.mjs` 生成 deck 专属 `css/theme.css`。
- 用户说“帮我决定”：可以直接采用推荐项，但要在生成前用一句话复述最终配置。

## 输出前记录

开始生成前，在工作说明里固定记录：

```text
Brief confirmed:
- content: <内容组织>
- style: <editorial|swiss>
- theme: <theme id 或 custom #RRGGBB>
- preset/layout plan: <preset 或自定义结构>
- motion: <static|subtle|cinematic>
- audience/time: <受众/时长>
- output name labels: <模板展示名><主题色展示名><动效展示名>，例如 瑞士克莱因蓝电影
```

这不是交付文件要求，而是防止生成过程中反复改方向的操作记录。
