# TrendRadar Prompt 模板

本文件定义了 TrendRadar 技能在不同阶段使用的 Prompt 模板，旨在引导 AI 更精准地理解用户意图并处理热点数据。

## 1. 主体拆解模板 (Subject Decomposition)

当用户输入一个模糊的想法或主体时，使用此模板将其拆解为适合搜索的关键词。

**角色**：热点分析专家
**任务**：将用户输入拆解为 3-5 个核心关键词。

**输入示例**：
> "我想了解下新能源汽车行业的最新趋势。"

**Prompt 内容**：
```markdown
请分析以下用户输入的主体或想法，提取出 3-5 个最能代表该领域热点动态的关键词。
输出要求：
1. 关键词应涵盖行业名称、核心技术、热点事件或代表性公司。
2. 以逗号分隔的纯文本格式输出，不要包含任何解释。

用户输入：{{user_input}}
```

---

## 2. 热点汇总与分析模板 (Trend Summary & Analysis)

当采集脚本返回 JSON 格式的热点列表后，使用此模板为用户生成易读的报告。

**角色**：资深舆情分析师
**任务**：根据采集到的热点数据，为用户生成一份结构清晰、有洞察力的热点周报。

**Prompt 内容**：
```markdown
以下是从多个平台（知乎、微博、抖音等）采集到的与“{{keywords}}”相关的热点内容：

{{raw_data}}

请根据以上数据执行以下任务：
1. **热点分类**：将热点按性质（如：技术突破、政策动向、市场动态、社会讨论）进行分类。
2. **列表展示**：每一项包含 [平台] 标题 - 链接。
3. **趋势总结**：简要分析目前该话题下全网最关注的 2-3 个核心点。

要求：
- 保持客观、简洁。
- 链接必须保留。
- 语言风格：专业且易懂。
```
